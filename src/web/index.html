Guide

使用说明

按实际使用顺序整理，方便第一次打开时快速上手。

01

先选择数据库

点击左侧数据源卡片，可以新建数据库，也可以切换到已有数据库。切换后页面会自动同步当前数据。

可选

导入 Excel / CSV

在数据库管理的新建数据库页选择 .xlsx、.csv 或 .tsv 文件后点击一键导入。第一行必须是表头，建议至少包含“公司名称”和“方向 / 岗位”。可识别字段包括公司名称、领域、企业性质、方向 / 岗位、城市、岗位性质、阶段、投递截止时间、投递链接、投递时间、笔试时间、笔试结果、面试时间和投递反馈。

02

添加投递信息

点击左侧“添加投递信息”，在弹窗中填写公司、岗位、阶段、日期和备注，确认后再保存写入数据库。

03

维护快捷入口

招聘信息和实用工具都支持自定义按钮。点击空白加号添加，点击按钮右上角图标可以编辑或删除。

04

查看总览与结构

总览页包含阶段分布、最近待办、近期更新和投递结构。点击任意条目可以弹出详情窗口查看对应公司列表。

05

筛选投递明细

投递明细页支持按阶段、领域、企业性质、城市和岗位性质筛选，也可以直接搜索公司、岗位和备注内容。

06

切换主题

右上角可以手动切换自动、白天和夜晚模式；自动模式会根据当前时间切换背景风格。
